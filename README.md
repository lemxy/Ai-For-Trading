# Ai-For-Trading
Autonomous ML trading signal system for 40 crypto &amp; stock assets. Stacking ensemble of LightGBM, XGBoost and RandomForest auto-tuned by Optuna. Purged Walk-Forward CV eliminates data leakage. FinBERT sentiment from 22 news sources, on-chain signals, 130+ technical indicators, dynamic SL/TP, Half-Kelly sizing. BUY/WAIT signals to Telegram 5× daily.
